# Cloud Infrastructure

Steps to onboard the Cloud Infrastructure.

## AWS Infrastructure

To onboard a AWS infrastructure, first visit the *Cloud Security* page. Select *Amanzon Web Services (AWS)* from the drop-down list, and then click the *Infrastructure* link in the upper-right corner of the page. Enter the required credentials and click *Create*.

## GCP Infrastructure
