# Dashboard

<Tabs>
  <TabItem value="dashboard" label="Dashboard" default>
    The dashboard enables a consolidated security view of all cloud infrastructure environments and infrastructure as code repositories. <p/>Before you begin, ensure that you have set up the cloud infrastructure environment and infrastructure as code repositories. <p/>The panels on the dashboards display various metrics computed based on the security audits. The dashboard provides multiple views to analyze the non-conformance based on different categories. These views help to analyze the non-conformance from different perspectives and drill down into details.

![](/img/docusaurus.png)

  </TabItem>
  <TabItem value="clouddashboard" label="Cloud Security">
    The Cloud Security dashboard provides details about non-compliances based on severity and by categories such as compliance standards, cloud accounts, resource types, and risk types. 
  </TabItem>
  <TabItem value="codedashboard" label="Code Security">
    The Code Security dashboard provides details about non-compliances based on severity and by categories such as platform and control types.
  </TabItem>
